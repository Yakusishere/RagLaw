# 12315 投诉流程说明

## 说明

- 资料用途：Phase 2 操作规则参考
- 资料性质：`rule`
- 主要来源：
  - `data/phase2/投诉流程图解/*.jpg`
  - `《中华人民共和国消费者权益保护法实施条例》第二十六条、第二十七条、第四十六条`

## 适用场景

- 商品质量问题
- 商家拒绝退款、退货、换货
- 平台内商家虚假宣传、售后拖延
- 需要行政投诉或调解介入的消费争议

## 平台入口

从全国 12315 平台进入投诉流程后，操作上通常会经历：

1. 选择 `我要投诉`
2. 选择投诉单位
3. 通过企业名称、统一社会信用代码或注册号搜索商家
4. 录入业务信息
5. 补充投诉内容与诉求
6. 上传附件后提交

## 提交前通常需要准备的信息

- 投诉对象名称
- 投诉对象住所地或经营地址
- 商品/服务名称
- 商品/服务类别
- 销售方式
  - 如：网购、现场购买
- 消费金额
- 争议发生时间
- 处理单位所在地
- 投诉问题类型
- 投诉内容
- 诉求内容
- 附件材料

## 投诉内容建议最少包含

- 购买时间
- 购买渠道
- 商品或服务名称
- 主要问题
- 与商家协商经过
- 法律依据或规则依据
- 具体诉求

## 诉求项建议

从用户截图看，常见诉求包括：

- 修理
- 重做
- 更换
- 退货
- 退赔费用
- 赔偿损失

其中对消费品质量纠纷最常见、最通用的是：

- `退赔费用`
- `赔偿损失`

## 附件建议

优先上传：

- 商品照片或问题视频截图
- 订单页面
- 支付记录
- 发票或收据
- 聊天记录
- 售后沟通截图

截图资料中提到一次上传附件数量有限，因此应优先保留最关键证据。

## 法律规则补充

根据 `《中华人民共和国消费者权益保护法实施条例》第四十六条`：

- 投诉应提供真实身份信息
- 应有明确的被投诉人
- 应有具体投诉请求和事实依据
- 有关行政部门应自收到投诉之日起 `7 个工作日内` 予以处理并告知消费者
- 受理后双方同意调解的，应在受理之日起 `60 日内` 调解完毕
- 需要鉴定、检测的时间不计入 60 日

## 对 `/draft` 的直接价值

这份规则可以直接反推出投诉信和平台投诉文案的结构化字段：

- `merchant_name`
- `merchant_address`
- `product_name`
- `product_category`
- `purchase_channel`
- `payment_amount`
- `issue_date`
- `issue_type`
- `complaint_content`
- `claim_items`
- `attachments`

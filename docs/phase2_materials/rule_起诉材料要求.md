# 起诉材料要求

## 说明

- 资料用途：Phase 2 操作规则参考
- 资料性质：`rule`
- 主要来源：`data/phase2/起诉需要材料.jpg`

## 截图整理后的材料清单

图片中给出的起诉所需材料，可以整理为以下结构：

### 1. 起诉状

- 需要提交 `1 份`
- 具状人需要本人亲笔签名
- 如果原告是公司，通常需要盖章

### 2. 当事人身份材料

#### 原告为个人

- 身份证复印件等
- 若涉及异地居住，还可能需要居住或经常居住证明材料

#### 原告为公司

- 营业执照
- 法定代表人身份证明
- 法定代表人身份证复印件

#### 代理人为公司员工时

- 授权委托书
- 劳动合同复印件
- 社保证明

### 3. 被告信息

#### 被告为个人

- 身份信息
- 户籍或住址相关信息

#### 被告为公司

- 企业工商登记信息
- 统一社会信用代码
- 公司住所地信息

### 4. 证据材料

- 需要提供证据目录清单
- 附件可包括：
  - 合同
  - 欠条
  - 银行流水
  - 聊天记录
  - 截图
  - 病历本
  - 发票
  - 其他与争议相关的证据

### 5. 送达地址确认书

- 需要填写并提交

## 对消费维权诉状的映射

如果把这份材料清单映射到网络购物纠纷场景，MVP 阶段最常见的提交材料应是：

- 起诉状
- 原告身份证明
- 被告主体信息
- 订单页面
- 支付记录
- 发票或收据
- 聊天记录
- 商品问题照片/视频截图
- 平台售后或投诉处理记录
- 送达地址确认书

## 对 `/draft` 的直接价值

- `missing_fields`
- `missing_materials`
